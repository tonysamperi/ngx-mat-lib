Utils

This page contains a set of useful components you may want to use in several points of your app.

These represent a fast way to obtain commonly used layouts, such as tables, summaries and others.

There also are some extra utils, like xmat-media-query-state, useful to debug your application with @angular/flex-layout (optional, required only if you want to use such component)
